# C_Study_Workspace

A Visual Studio (MSVC) workspace for my C programming coursework: lectures, exercises, and small demos.

## Toolchain
- Windows 11
- Visual Studio 2022 (MSVC)
- C (C11-style learning; MSVC-compatible)
- Input via `scanf_s` (validated)

## Layout
- Each folder is an independent VS project (`.vcxproj`) with its own `main.c`
- Projects cover core fundamentals (control flow, functions, input validation, small interactive apps)
